📖 Guide — Supabase Edge Function
Déployer le pont SMS en 3 étapes :
Étape 1 — Créer la function :
supabase functions new send-sms
Étape 2 — Code index.ts (Orange) :
import { serve } from "std/server"
serve(async (req) => {
  const { to, message } = await req.json()
  const apiKey = Deno.env.get('ORANGE_API_KEY')
  const sender = Deno.env.get('SMS_SENDER_ID')
  const res = await fetch(
    'https://api.orange.com/smsmessaging/v1/outbound/'+
    encodeURIComponent('tel:'+sender)+'/requests',
    { method:'POST',
      headers:{ Authorization:'Bearer '+apiKey,
                'Content-Type':'application/json' },
      body: JSON.stringify({ outboundSMSMessageRequest: {
        address: 'tel:'+to,
        senderAddress: 'tel:'+sender,
        outboundSMSTextMessage: { message }
      }})
    })
  const data = await res.json()
  return new Response(JSON.stringify(data))
})
Étape 3 — Variables d'environnement :
supabase secrets set ORANGE_API_KEY=xxx
supabase secrets set SMS_SENDER_ID=ALLTEC
supabase functions deploy send-sms
